Sinkronisasi ke Mekari Jurnal

Siapkan token API Mekari Jurnal untuk otomatisasi posting penjualan dan jurnal.

1

Hubungkan Integrasi

Buka menu Integrasi Aplikasi API dan tambahkan kredensial Mekari Jurnal untuk mengaktifkan sinkronisasi otomatis.

2

Atur Skema Akun

Mapping akun penjualan, HPP, dan piutang agar data yang dikirim ke Mekari Jurnal langsung siap dibukukan.

3

Sinkron Otomatis

Gunakan tombol Sinkronkan Sekarang atau jadwalkan sinkronisasi berkala untuk memastikan laporan selalu terbaru.
